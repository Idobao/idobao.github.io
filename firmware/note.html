Note

There is some confusion around versions.

The trick to understanding is that there are two main components (that have version numbers) of a keyboard:

  1. The Case
  2. The PCB

The Case

The version of the case is the one that is most often referred to. All the marketing collateral will cite the case version.

The PCB

The PCB is where all the electronics and the MCU live.

It is not uncommon for PCB and Case versions to be misaligned. (This is normal.) Some times new PCB's are installed in existing cases (e.g., V2 case and V3.1 PCB ), and sometimes the same PCB can be in more than one case (e.g., V2 PCB in cases A, and B).

WARNING: Always make sure to install the correct PCB version of the firmware.
