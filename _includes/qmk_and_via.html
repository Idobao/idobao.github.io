IDOBAO keyboards are built on QMK firmware.

You can always use the QMK Configurator ( config.qmk.fm)NB, but note that firmware from there does not include the VIA code library.

NB = QMK Configurator may not have all the variants available.

All our keybards are programmed with the VIA variant of the code. If you need to restore these please use the links below.

Be sure to read our Manuals if you need help on that.
